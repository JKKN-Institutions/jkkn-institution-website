## âš¡ Common Tasks

### Add a New Benefit
1. Go to "**Why Choose**" tab
2. Click "**+ Add Benefit**"
3. Fill in: Icon (emoji), Title, Description
4. Click "**Save Changes**"

### Update Recruiters List
1. Go to "**Recruiters**" tab
2. Click "**+ Add Company**" to add new
3. Click trash icon to remove companies
4. Click "**Save Changes**"

### Add Faculty Member
1. Go to "**Faculty**" tab
2. Click "**+ Add Faculty**"
3. Fill in: Name, Designation, Qualification, Specialization, Image URL
4. Click "**Save Changes**"

### Update Fee Structure
1. Go to "**Fee Structure**" tab
2. Click "**+ Add Component**" for new fee items
3. Fill in: Component name, Amount
4. Check "**Is Total Row**" for total/sum rows
5. Click "**Save Changes**"

### Add FAQ
1. Go to "**FAQs**" tab
2. Click "**+ Add FAQ**"
3. Fill in: Question, Answer
4. Click "**Save Changes**"

### Change Brand Colors
1. Go to "**Colors & Styling**" tab
2. Click color picker or enter hex code
3. **Primary Color**: Main green (#0b6d41)
4. **Accent Color**: Highlight orange (#ff6b35)
5. Click "**Save Changes**"

# Page Templates

This reference provides ready-to-use page templates for common page types. Each template includes a pre-configured set of components that can be customized after adding.

---

## Template Categories

1. **Marketing** - Landing pages, campaigns
2. **Informational** - About, history, policies
3. **Academic** - Programs, departments, courses
4. **Event** - Events, webinars, conferences
5. **Media** - Galleries, videos, press
6. **Contact** - Contact, support, locations

---

## Marketing Templates

### 1. Landing Page Template
**Best for:** Marketing campaigns, program promotions, admissions

**Structure:**
```
1. Hero Section (full height)
   - Background image with overlay
   - Main headline + subtitle
   - Primary CTA button + Secondary button

2. Stats Counter
   - 4 key statistics with icons
   - Animated count-up

3. Features Grid (3 columns)
   - Section heading
   - 3 feature cards with icons

4. Testimonials Carousel
   - Section heading
   - 3+ student/parent testimonials
   - Auto-rotating carousel

5. FAQ Section
   - Common questions
   - Expandable accordion

6. CTA Banner (full width)
   - Compelling headline
   - Description
   - Primary action button
```

**Content to prepare:**
- Hero image (1920x1080 minimum)
- 4 statistics with numbers
- 3 feature titles with descriptions
- 3+ testimonials with photos
- 5-10 FAQ items
- Final CTA text and link

## Template Usage Instructions

### How to Use a Template

1. **Create New Page**
   - Go to `/admin/content/pages`
   - Click "Create Page"
   - Enter page title and slug

2. **Apply Template**
   - In page editor, click "Templates" button
   - Browse by category
   - Click "Use Template"

3. **Customize Content**
   - Click each component
   - Update text, images, links
   - Add or remove sections as needed

4. **Preview and Publish**
   - Toggle preview mode
   - Check mobile view
   - Publish when ready

### Customizing Templates

**Adding sections:**
- Drag new components from palette
- Drop at desired position

**Removing sections:**
- Click section to select
- Press Delete or click trash icon

**Reordering sections:**
- Drag section by handle
- Drop at new position

**Changing backgrounds:**
- Select Section Wrapper
- Update background property

## Template Best Practices

### Content Preparation Checklist

Before using a template, prepare:

- [ ] Page title and URL slug
- [ ] Hero image (1920x1080+)
- [ ] Main heading and subheading
- [ ] Body text content
- [ ] CTA button text and links
- [ ] Supporting images
- [ ] Testimonials (if needed)
- [ ] FAQ items (if needed)

### Image Guidelines

| Location | Recommended Size | Format |
|----------|-----------------|--------|
| Hero background | 1920x1080 | JPG, WebP |
| Gallery images | 800x600 | JPG, WebP |
| Team/faculty photos | 400x400 | JPG, PNG |
| Logos | 200x100 | PNG, SVG |
| Icons | 64x64 | SVG, PNG |

### SEO Checklist

For each page:
- [ ] Meta title (under 60 characters)
- [ ] Meta description (under 160 characters)
- [ ] Alt text for all images
- [ ] Meaningful heading hierarchy (H1 → H2 → H3)
- [ ] Internal links to related pages

## Creating Custom Templates

To save a page as a new template:

1. Build the page with all components
2. Configure all sections
3. Click "Save as Template" in toolbar
4. Enter template name and category
5. Add thumbnail image
6. Save

The template will be available for future pages.

---

## Template Maintenance

### Updating Templates
When templates need updates:
1. Create new page from template
2. Make required changes
3. Save as new template (same name)
4. Old template is replaced

# 2018-2019 Feedback PDFs

## Required PDF Files

Please add the following PDF files to this directory:

1. **student-feedback-2018-2019.pdf** - Student feedback report
2. **alumni-feedback-2018-2019.pdf** - Alumni feedback report
3. **parents-feedback-2018-2019.pdf** - Parents feedback report
4. **faculty-feedback-2018-2019.pdf** - Faculty feedback report

## File Naming Convention

- Use lowercase with hyphens
- Include the academic year in the filename
- Format: `{category}-feedback-2018-2019.pdf`

## How to Add Files

1. Copy your PDF files to this directory
2. Ensure the filenames match exactly as listed above
3. The page will automatically link to these PDFs

# NIRF Documents

This directory contains NIRF (National Institutional Ranking Framework) ranking reports.

## Directory Structure

```
nirf/
├── 2024/
│   ├── engineering.pdf
│   ├── management.pdf
│   ├── innovation.pdf
│   └── overall.pdf
└── 2025/
    ├── engineering.pdf
    ├── management.pdf
    ├── innovation.pdf
    ├── overall.pdf
    └── sdg.pdf
```

## Adding PDF Files

1. Place the actual NIRF PDF reports in the respective year folders
2. Ensure filenames match exactly as listed above (lowercase)
3. Files will be accessible at `/documents/nirf/{year}/{category}.pdf`

## NIRF 2024 Categories

- **Engineering** - Engineering rankings report
- **Management** - Management rankings report
- **Innovation** - Innovation rankings report
- **Overall** - Overall institutional rankings report

## NIRF 2025 Categories

- **Engineering** - Engineering rankings report
- **Management** - Management rankings report
- **Innovation** - Innovation rankings report
- **Overall** - Overall institutional rankings report
- **SDG** - Sustainable Development Goals rankings report (New in 2025)

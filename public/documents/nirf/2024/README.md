# NIRF 2024 Documents

This directory contains the NIRF 2024 ranking reports for JKKN Institutions.

## Required Files

Please place the following PDF files in this directory:

1. **engineering.pdf** - Engineering category rankings
2. **management.pdf** - Management category rankings
3. **innovation.pdf** - Innovation rankings
4. **overall.pdf** - Overall institutional rankings

## File Naming Convention

- Use lowercase filenames
- Use hyphens for multi-word names (if needed)
- Keep filenames simple and descriptive

## Notes

- All files should be in PDF format
- Recommended file size: Under 5 MB for optimal loading
- Ensure PDFs are optimized for web viewing
- Update the file sizes in the data files after uploading actual PDFs
